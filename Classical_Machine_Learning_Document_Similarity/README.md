# Classical Machine Learning for Document Similarity

This project demonstrates how **Classical Machine Learning** techniques can be used to measure document similarity and perform document retrieval. It explores multiple similarity measures, including **Jaccard Similarity**, **Cosine Similarity**, and **TF-IDF**, along with **K-Nearest Neighbors (KNN)** for finding related documents.

---

## Overview

Before the rise of modern deep learning models, document retrieval and text similarity were commonly solved using feature engineering and vector-space models. This notebook provides practical implementations of these classical techniques and compares their effectiveness in identifying similar documents.

---

## Features

- Jaccard Similarity implementation
- Cosine Similarity computation
- TF-IDF vectorization
- K-Nearest Neighbors (KNN) for document retrieval
