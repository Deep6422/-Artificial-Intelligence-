# Semantic Network

A Python implementation of a **Semantic Network**, demonstrating knowledge representation techniques used in Artificial Intelligence. This project models concepts as nodes and relationships as edges, allowing users to visualize and explore how knowledge can be organized and connected.

---

## Overview

Semantic Networks are one of the earliest knowledge representation techniques in Artificial Intelligence. They represent objects, concepts, and entities as **nodes**, while the relationships between them are represented as **edges**.

This notebook demonstrates how semantic networks can be used to represent knowledge, perform relationship analysis, and visualize interconnected concepts.
