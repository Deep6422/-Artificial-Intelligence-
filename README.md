# Artificial Intelligence

A collection of **Artificial Intelligence laboratory projects** implemented in **Python** using **Jupyter Notebook**. Each project focuses on a fundamental AI concept, including knowledge representation, search algorithms, machine learning, and intelligent problem-solving.

This repository is continuously updated as new AI projects and laboratory exercises are completed.

---

## About

This repository contains practical implementations of core Artificial Intelligence concepts learned through laboratory work and personal exploration.

Each project is organized in its own folder and includes:

- 📓 Jupyter Notebook implementation
- 📄 Project documentation (`README.md`)
- 💡 Theory and explanations
- 🧪 Practical examples and demonstrations

---

## Current Projects

| Project | Description |
|---------|-------------|
